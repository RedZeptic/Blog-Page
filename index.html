Day 018: August 18th 2022:

I started working on a Balance Sheet (not done, 50% there) page today using html tables and setting it up for screen reader help. Here are some of the elements I used today, ones I'm still new to using:

HTML

  • < table >, the tag responsible for building out a table in html. Similar to any container tag, but only for tables.
  • < caption >, this tag is required to be the first in the table tag, and defines a text caption for the table.
  • < thead > (table head), this tag is similar to the header tag, but for the table container tag.
  • < tbody > (table body), this tag is in the same position as the header but replicating the body container for the table.
  • < tr > (table row), this defines a row in an HTML table.
  • < th > (table header), this tag defines a header cell in an HTML table.
  • < td > (table data), this defines a header cell in an HTML table, essentially to a label for the column.

CSS

  • span[class~="sr-only"], using this in place of .sr-only and I'm not really sure why this way would be better.
  • clip: Attribute.
  • clip-path: Attribute.
  • -webkit-clip-path: Attribute, which confuses me currently.
  • inset(50%) value.
  • overflow: Attribute.
  • white-space: Attribute.
  • nowrap; value. relating to the white-space above.
