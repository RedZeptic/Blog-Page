Anyhow after watching videos on this process and building out my form, I went around this site and adjusted some features that were inconsistant on some pages, as well as added new ones!

Which leads me right into what I worked on for the majority of my time spent today, which was around 1.5-2hrs. Started up the next project which was all about making a Cafe Menu page and styling it with CSS, you can check out my progress here. Its SUPER simple as of now but this is about as far as I could get with the time I had allocated today. And I was familiar with the vague layout/format of CSS, which comes from way back in 2018 when I was exploring web dev education for my first time. I use to watch a lot of a youtuber named Stefan Mischook on his YouTube page, and actually bought a course from his website to learn how to code HTML, CSS and JS. I'll say that now reflecting on that is fuzzy but the underlying "brickwork" is there as my starting foundation and I feel this wont be as difficult as it could have been if I was still a complete novice.
